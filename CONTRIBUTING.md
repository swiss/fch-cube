## How to contribute to fch-cube?

#### **Did you find a bug?**

* **Do not open up a GitHub issue if the bug is a security vulnerability in fch-cube-****. Please follow the procedure described in 
[SECURITY.md](./SECURITY.md).

* **Ensure the bug was not already reported** by searching on GitHub under [Issues](https://github.com/swiss/fch-cube/issues).

* If you're unable to find an open issue addressing the problem, [open a new one](https://github.com/swiss/fch-cube/issues/new). Be sure to include a **title and clear description**, as much relevant information as possible, and a **code sample** or an **executable test case** demonstrating the expected behavior that is not occurring.

#### **Did you write a patch that fixes a bug or want to add new functionality?**

* This project is currently not under a community development phase, please create an GitHub issue first to discuss improvements to the current code base.
